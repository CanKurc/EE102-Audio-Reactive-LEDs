## 📌 Project Overview
This repository contains the complete hardware and VHDL implementation for a standalone, music-reactive LED controller built on the AMD Xilinx Artix-7 FPGA (Basys 3). 

The system continuously acquires raw analog audio from a room microphone, processes the signal using a custom discrete-time Moving Average DSP filter to isolate bass frequencies, and utilizes the resulting data to drive a highly precise, nanosecond-accurate WS2812B addressable LED strip.

**Strict Engineering Constraint:** This architecture relies on 100% pure VHDL. It strictly avoids microcontrollers, software interrupts, soft-core processors (like MicroBlaze), and AXI4-Lite IP blocks. All logic is cycle-accurate and hardware-deterministic.

## 🛠️ Hardware Architecture & Safety 

### Core Processing
* **FPGA:** Digilent Basys 3 (Artix-7 XC7A35TCPG236-1).
* **System Clock:** 100 MHz internal oscillator (10ns period).

### Analog Front-End (AFE)
* **Sensor:** MAX4466 Analog Sound Sensor (Powered via Basys 3 3.3V rail).
* **XADC Protection:** The Basys 3 JXADC header enforces a strict 0V to 1.0V maximum input range. A custom passive voltage divider utilizing 10kΩ and 4.7kΩ 1% resistors safely steps the 3.3V MAX4466 output down to ~1.0V.
* **Routing:** Analog signal routed to `VAUX6` via JXADC Pin 1.

### High-Current Power Delivery
* **LEDs:** Hazar WS2812B Addressable ARGB Strip (60 LEDs/meter).
* **External Power:** Mervesan MT-60-5 Industrial SMPS (5V, 12A, 60W).
* **Safety & Grounding:** The Mervesan PSU is connected to 220V AC with a mandatory Earth Ground bolted to the chassis. The DC Ground is tied directly to the Basys 3 GND (Header JA, Pin 5) to establish a shared reference plane for the data signal.
* **Decoupling:** A 1000µF 25V electrolytic capacitor is wired across the 5V and GND rails at the LED injection point to absorb massive transient inrush currents.

### Signal Integrity
* **Impedance Matching:** A 330Ω discrete resistor is soldered in series with the WS2812B DIN line to dampen high-frequency transmission line reflection ("ringing") generated by the FPGA's nanosecond edges.

## 💻 VHDL Modules & System Logic

### 1. XADC Instantiation (`xadc_wiz_0`)
The on-chip Analog-to-Digital Converter is configured via the Vivado IP Catalog utilizing the Dynamic Reconfiguration Port (DRP) in Continuous Sampling Mode to capture 12-bit audio data without external trigger logic.

### 2. Physical Layer LED Driver (`led_driver.vhd`)
A custom Finite State Machine (FSM) translates the 100MHz system clock into the single-wire 800kHz Return-to-Zero (RZ) protocol required by the WS2812B.
* **Logic 0:** 400ns High (40 cycles) / 850ns Low (85 cycles).
* **Logic 1:** 800ns High (80 cycles) / 450ns Low (45 cycles).
* **Latch/Reset:** >300µs line low to commit the video frame.

### 3. Digital Signal Processing (`moving_average.vhd`)
To avoid exhausting DSP slices with hardware division, the audio is smoothed using a discrete-time Moving Average Filter. A circular FIFO of 12-bit shift registers stores recent samples. The sum of these registers is bitwise right-shifted (`Total_Sum(13 downto 2)`), elegantly performing a hardware divide-by-4.

### 4. Top-Level Controller (`top_module.vhd`)
The top module instantiates all components, compares the filtered DSP output to physical threshold switches, and increments a color automation state machine upon detecting a bass beat.
